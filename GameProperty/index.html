How are monsters created and added to the scene ?üìå

The main goal in the R-type game is to kill the monsters present on the screen. However, the monsters representing the enemies are created using the Rtypefactory::newMonster method via the scene which generates a number for the new entity to which properties such as position, speed, and archetype (monster) are associated. Then the movement of the monster is coordinated in the game loop via a variable.

How are the different ships added in the scene ? üïπÔ∏è

Ships are added to the scene with a CatchEntity method which uses an entity template. This is one of the features of the game engine.

How is data sent from the server to the client? üß§

When the data is too heavy they are compressed using the DEFLET algorithm which combines the Hufman algorithm and that of AL77. We then serialize the data. Each piece of information is associated with the id of a command which can be a request, a response or a notification.

How can you implement a game with the CHAOS Engine ? üîñ

In the game engine we had to encapsulate the Raylib which represents the library we use. As a result, the developer will have to be careful to use the methods encapsulated at the Game Engine level in order to develop their game efficiently.
